# Images Directory

This directory contains images, diagrams, and visual assets for the ePortfolio.

## Recommended Contents

### Architecture Diagrams
- `system-architecture.png` - Overall system architecture
- `esp32-architecture.png` - ESP32 firmware component diagram
- `django-architecture.png` - Django backend architecture
- `database-erd.png` - Entity-Relationship Diagram

### State Machine Diagrams
- `thermostat-state-machine.png` - Thermostat control states
- `event-flow.png` - Event handling flowchart

### Screenshots
- `dashboard-screenshot.png` - Web dashboard preview
- `lcd-display.png` - LCD display output
- `hardware-setup.jpg` - Physical hardware setup

### Algorithm Visualizations
- `hysteresis-graph.png` - Hysteresis control visualization
- `algorithm-flowchart.png` - Control algorithm flowchart

## Image Guidelines

- Use PNG format for diagrams and screenshots
- Use JPG format for photographs
- Recommended width: 800-1200px for optimal web display
- Include alt text descriptions in HTML for accessibility
