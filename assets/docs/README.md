# Documents Directory

This directory contains PDF documents and narrative files for the ePortfolio.

## Required Documents

### Narratives
- `software-engineering-narrative.pdf` - Software Engineering enhancement narrative
- `algorithms-narrative.pdf` - Algorithms & Data Structures enhancement narrative
- `database-narrative.pdf` - Databases enhancement narrative

### Supporting Documents
- `code-review-transcript.pdf` - Code review video transcript (accessibility)
- `project-documentation.pdf` - Technical documentation for the project
- `resume.pdf` - Professional resume (optional)

## Document Guidelines

- All narratives should follow CS-499 rubric requirements
- Include proper citations and references
- PDFs should be web-optimized for faster loading
- Consider providing HTML versions for better accessibility

## Narrative Structure

Each enhancement narrative should include:
1. Artifact description and origin
2. Justification for inclusion
3. Reflection on enhancement process
4. Course outcomes addressed
5. Challenges and solutions
